\begin{abstract}
Agentic AI systems that autonomously reason, use tools, and execute multi-step workflows are transforming enterprise AI deployment. However, enterprise environments introduce challenges absent from single-tenant academic treatments: multiple tenants with heterogeneous data, strict access controls, regulatory compliance, and cost pressures demanding shared infrastructure.

A fundamental security gap underlies existing agentic architectures: dense retrieval ranks documents by semantic similarity, not authorization, enabling cross-tenant data leakage when agents retrieve similar but unauthorized content. We formalize this similarity-authorization gap and analyze additional failure modes unique to agentic systems. We introduce a layered isolation architecture combining policy-aware ingestion, retrieval-time gating, and shared inference, enforced through server-side agentic orchestration.

Our open-source implementation using Llama Stack demonstrates that secure multitenancy, cost-efficient resource sharing, and autonomous agent capabilities are simultaneously achievable on shared infrastructure, enabling enterprise deployment of agentic AI with systematic behavioral control.
\end{abstract}
